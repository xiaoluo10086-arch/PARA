# Curated Results

This directory contains the compact evidence retained in the public Git
repository:

- `summaries/`: paper-facing aggregate results and matched baselines.
- `proof_audit/`: sampled `SUPPORTED` and `INCONCLUSIVE` traces plus audit
  reports.
- `accountability/`: counterfactual and structurally matched near-miss tests.
- `project_specific_policy/`: the Spring project-specific policy case summary.

Paths beginning with `artifacts/` refer to files in the separately distributed
full artifact bundle. They are provenance references, not missing Git files.

Invalid transport runs, debug probes, superseded result versions, and large
intermediate directories are intentionally excluded.
