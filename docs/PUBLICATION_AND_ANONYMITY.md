# Publication and Review Anonymity

This repository is the official named public home for PARA.

For a double-anonymous review process:

1. Do not link this named repository from the anonymized manuscript.
2. Create a separate anonymous artifact snapshot with identical source and data.
3. Remove author names, account names, repository URLs, and identifying commit
   metadata from that review snapshot.
4. After the review period, point the paper and archival artifact to this
   official repository.

The public and anonymous snapshots should be generated from the same reviewed
commit so their code and result hashes match.
