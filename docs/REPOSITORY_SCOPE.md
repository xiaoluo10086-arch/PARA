# Public Repository Scope

## Include

- Final PARA implementation.
- Scripts required to reproduce reported experiments.
- Frozen, valid paper summaries and sampled proof traces.
- Small self-contained example data.
- Figure source data and plotting code.
- Installation, data, and reproducibility documentation.

## Exclude

- Secrets, `.env` files, tokens, private endpoints, and account identifiers.
- Local absolute paths.
- Model weights and large caches.
- Third-party repositories and compiled build trees.
- Obsolete code branches, abandoned experiments, and invalid API runs.
- Full manuscript history or internal review notes.
- Files whose redistribution license is unclear.

Large valid artifacts are release assets, not Git history.
